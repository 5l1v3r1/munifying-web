This PoC is meant to extract AES keys from vulnerable Unifying receivers (CVE-2019-13054, CVE-2019-13055). It only works for unpatched receivers with TI chip and uses upcoming WebHID API.

Currently WebHID if available under a flag in Chrome 78 and supposed to arrive with Chrome 80. WebHID could be enabled using the 'Experimental Web Platform features' here: chrome://flags/#enable-experimental-web-platform-features




LOGITacker commands to store devices along with keys (for live decryption)
